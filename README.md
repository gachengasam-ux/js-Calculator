# JS Calculator

A simple, interactive calculator built with **JavaScript**, **HTML**, and **CSS**. This project demonstrates basic DOM manipulation, event handling, and arithmetic operations in the browser.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division  
- Clear and reset functionality  
- Responsive layout for desktop and mobile  
- User-friendly interface
